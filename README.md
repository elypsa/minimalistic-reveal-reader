# ChapterPal Lite

A minimalist progressive reveal reader for distraction-free reading of markdown files.

## Features

- **Progressive Reveal**: Read content sentence-by-sentence or paragraph-by-paragraph
- **Customizable Reading Experience**: Adjust font size, line height, and viewing context
- **Multiple Themes**: Light, Sepia, and Night reading modes
- **Chapter Navigation**: Automatic chapter detection from markdown headings
- **Progress Tracking**: Visual progress bar and reading statistics
- **Markdown Support**: Full markdown formatting including emphasis, code, and links

## Usage

1. Open `reader.html` in your web browser
2. Select a markdown file to read
3. Use **Space** or **Right Arrow** to advance
4. Use **Left Arrow** to go back
5. Press **Escape** to return to file selection
6. Click the **⚙** icon to customize settings

## Keyboard Shortcuts

- `Space` / `Right Arrow` - Next sentence/paragraph
- `Left Arrow` - Previous sentence/paragraph
- `Escape` - Return to file picker

## Customization

Access the settings panel to adjust:
- Font size (14-32px)
- Line height (1.2-2.5)
- Number of past sentences visible (1-10)
- Reading style (Sentence or Paragraph mode)
- Color theme (Light, Sepia, or Night)

## Local Storage

The app automatically saves:
- Your reading position
- Personal settings preferences

All data is stored locally in your browser.
